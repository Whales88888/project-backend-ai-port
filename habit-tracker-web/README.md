# Habit Tracker Web Application

## 概要
日常の習慣を可視化し、継続を支援することを目的とした  
シンプルな Web アプリケーションです。  
ユーザーが習慣を登録し、進捗状況を直感的に確認できる構成としています。

---

## 目的
- 習慣の継続状況を分かりやすく管理する
- サーバーに依存しない軽量な Web アプリを設計・実装する
- フロントエンド開発の基礎技術を実装ベースで習得する

---

## システム構成
- フロントエンドのみで完結する構成
- データはブラウザの localStorage に保存
- PC / スマートフォン両対応（レスポンシブ対応）

---

## 工夫した点
- サーバー不要とすることで、環境構築の手間を削減
- JavaScript による DOM 操作を整理し、可読性を重視
- UI はシンプルさと操作性を優先して設計
- 機能単位で処理を分け、将来的な拡張を想定した構成

---

## 使用技術
- HTML5 / CSS3
- JavaScript
- localStorage
- Git / GitHub

---

## 学習ポイント
- フロントエンド開発の基本構造理解
- DOM 操作およびイベント処理
- クライアントサイドでのデータ管理
- UI/UX を意識した画面設計

---

## 実行方法
1. 本リポジトリを clone
2. `BSHabitTracker.html` をブラウザで開く
3. 習慣を入力し、進捗管理が可能
