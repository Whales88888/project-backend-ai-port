Pet Clinic Management System

---

## 概要
本プロジェクトは、動物病院における日常業務を効率化することを目的として開発した  
Web ベースの管理システムです。

ペット情報、飼い主情報、診療・予約情報を一元的に管理し、  
現場業務の負担軽減と情報管理の正確性向上を目指しました。

---

## 開発目的
- 動物病院の業務フローを理解し、システムとして整理すること  
- 実務を想定した Web アプリケーション設計・実装の経験を積むこと  
- バックエンドを中心としたシステム開発の基礎力を身につけること  

---

## 主な機能
- ペット情報の登録・編集・削除  
- 飼い主情報の管理  
- 診療・予約情報の管理  
- データベースによる情報の一元管理  

---

## システム構成
- バックエンドを中心とした Web アプリケーション構成  
- データはデータベースで管理  
- 小規模システムとしての運用を想定  

---

## 使用技術
- Python  
- Flask  
- MySQL  
- HTML / CSS  
- Git / GitHub  

---

## 工夫した点
- 実際の業務を想定し、データ項目や画面構成を設計  
- 機能ごとに処理を整理し、可読性と保守性を重視  
- 将来的な機能追加を想定したシンプルな構成  

---

## 学んだこと
- 業務要件をシステムに落とし込む考え方  
- CRUD 処理とデータベース設計の基礎  
- バックエンド開発を中心とした Web アプリケーション構築
